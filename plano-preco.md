## 1) O produto

Um ecossistema com 10 categorias de ferramentas:

| Categoria | Exemplos |
|-----------|----------|
| 📄 PDF | Juntar, dividir, comprimir, converter, OCR, assinar, marca d'água, resumir com IA |
| 🖼️ Imagens | Comprimir, redimensionar, converter, remover fundo, upscale, favicon, paletas |
| 🔐 Segurança | Gerador/cofre de senhas, hash, QR Code, criptografia, JWT, UUID, 2FA |
| 🤖 IA | Resumir, traduzir, reescrever, corrigir, gerar (e-mail/contrato/proposta), currículo |
| 🌐 Web | Screenshot, minificar, robots/sitemap, SEO, SSL, DNS, WHOIS, uptime |
| 🧰 Dev | JSON/XML/YAML, Base64, URL, regex, SQL, gerador de dados/CPF/CNPJ, Markdown |
| 📊 Escritório | CSV↔Excel, mesclar/dividir planilhas, contadores, certificados, contratos |
| 🎥 Áudio/Vídeo | Converter, comprimir, cortar, extrair áudio, legendas, transcrição |
| 💰 Finanças | Conversor de moedas, calculadoras (juros/empréstimo/investimento/impostos), boletos |

A maioria das ferramentas roda **no navegador** (privacidade — o arquivo não sai do dispositivo);
as de **IA** usam o modelo da plataforma; as **premium** liberam lote, API e integrações.
